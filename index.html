Frequently Asked Questions

Are these OSHA kits specific to HVAC jobsite requirements?

Yes — all recommended kits meet OSHA standards and are curated with HVAC-specific mobility, PPE, and first-aid needs in mind.

What’s the difference between Class A and Class B kits?

Class A kits are for low-risk environments, while Class B kits — ideal for HVAC — include more comprehensive supplies for moderate to high-risk jobsites.

Do I need a kit in each HVAC truck?

OSHA recommends a compliant first aid kit be available where employees work — so equipping each vehicle ensures you're covered during mobile jobs.
